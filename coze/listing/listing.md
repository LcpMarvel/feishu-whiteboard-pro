## 技能名称（4–6 字，≤10）
飞书白板大师

## 一句话简介（≤30 字）
一句话把内容生成为有设计感、可编辑的飞书白板

> 备选：「为飞书生成真正设计过的可编辑白板，而非方框网格」

## 详细介绍

**它解决什么**
普通"AI 画白板"往往是一堆等大方框的网格，平、挤、没重点。本技能不一样：它在动笔前先定**设计简报**
（构图原型 + 配色策略 + 字号角色 + 反套路检查），按**坐标骨架 + 固定字号 + 间距栅格**施工，画完再跑
一遍**设计评审**（层级 / 平衡 / 密度 / 对比 / 对齐），补到最弱项达标——产出的是有焦点、有层级、会呼吸
的版面，而且是写进你自己飞书文档里的**真实可编辑白板**，不是一张截图。

**核心功能**
1. **构图原型库**：序列 / 对比 / 层级 / 放射 / 矩阵 / 时间线 / 焦点+细节等，按内容结构自动选型并套坐标骨架。
2. **35 套精选配色**：按气质与正式度选色，配色策略分克制 / 投入 / 全色三档。
3. **渲染前预检 + 渲染后评审**：文字溢出 / 出血先于渲染被预测；五轴评审挑出最弱项并给修法。
4. **写进你自己的飞书**：以你本人身份建文档 + 可编辑白板，产出飞书文档链接 + 渲染图。

**怎么用**
- 直接给内容或主题，例如：「把这套增长策略做成一张飞书白板」「用白板讲清楚我们的系统架构」。
- 技能自动选原型与配色、生成、自检；首次使用时会给你一个**授权链接，浏览器点一下同意**即可
  （之后免授权复用）。
- 完成后给你**飞书文档链接**和**预览图**；你可随时让它**换一套配色**，同构图重渲。

**注意事项**
- 需要**飞书 / Lark 账号**；白板写进你**自己的**租户、归你所有。
- 首次需一次性浏览器授权（无需你创建应用、无需填任何密钥）。
- 白板媒介刻意受限：单字体、原生矩形/圆/连接线，**无渐变 / 滤镜 / 透明度 / 动效**——"好看"来自构图与
  留白，不靠特效。
- 导出图对版面/填充忠实，但**文字颜色**以实时飞书文档为准。
